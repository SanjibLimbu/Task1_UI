## Run Locally

Clone the project

```bash
  git clone https://github.com/SanjibLimbu/Task1_UI.git
```

Go to the project directory

```bash
  cd Task1_UI
  cd UI
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```
